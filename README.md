# HayDay Profit Guide

A fan-made guide for the mobile game Hay Day, focused on helping players make the most profit. This app provides sortable tables for crops, trees and bushes, and Tom's requests so you can easily see what's worth growing, planting, or requesting from Tom.

## Stack
- Framework: React 19
- Build tool: Vite 8
- Styling: Tailwind CSS v4
- Languages: JavaScript, HTML, CSS
- Data: Local JSON files

The app is available at https://pepppa9351.github.io/HayDayProfitGuide/


*This is an unofficial fan project and is not affiliated with Supercell or the Hay Day brand.*
